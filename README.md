# Ejercicio_Semana_1
Esta tarea consiste en un scraper desarrollado en Python.

## Descripción de la tarea
El programa accede a una página web de noticias y extrae información de 5 noticias.

## Datos extraídos en la tarea
- Título de la noticia
- Enlace
- Autor
- Fecha

## Tecnologías utilizadas
- Python
- Requests
- BeautifulSoup
